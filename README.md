# LawSage-Server

## Run Locally

Clone the project

```bash
git clone https://github.com/adityaxp/LawSage-ServerV0.2.git 
```

Install Dependencies

```bash
pip3 install -r requirements.txt
```

LLM Service

```bash
cd LLMService
python3 LLMService.py
```

RAG Service

```bash
cd RAGService
python3 RAGService.py
```
